# awesome-sentence-embeddings
A collection of sentence embeddings literature.

The list is based on a great survey: [A Comprehensive Survey of Sentence Representations: From the BERT Epoch to the CHATGPT Era and Beyond](https://aclanthology.org/2024.eacl-long.104/), combined with other stuff I found. The goal of this list is to **maximize recall**.
